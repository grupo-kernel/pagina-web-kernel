# Grupo de Investigación El Kernel

Sitio institucional y laboratorio estadístico del Grupo El Kernel. La
aplicación se construye con Vite y ejecuta las calculadoras localmente en el
navegador; los datos introducidos en ellas no se envían a un servidor para
realizar los cálculos.

## Desarrollo

Requiere Node.js 22.

```bash
npm ci
npm run dev
```

Validación completa:

```bash
npm run test:all
npm run build
npm run test:dist
```
